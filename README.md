# CS6461
the computer_simulator project for CS6461



### IPL.txt

**program:**

LDR 0,0,14

STR 0,0,15

LDX 0,1,15

STX 0,1,16

LDA 1,1,20

LDR 2,1,0,1

STR 2,1,20,1

Attention: the program counter will start at 6, so the first instruction of program must be written to memory at address 6.

**Data:**

MEM[14] = 20

MEM[20] = 40

MEM[40] = 80

Attention: all the numbers here are digital
